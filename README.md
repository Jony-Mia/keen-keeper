<h1>Keen Keeper</h1>
<p>Keen Keeper is a friendship management app. Where use can see their friends. And their profile based on data. User can contact your friends without hesitating </p>

<h2> Technology used to built it </h2>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>Tailwind Css</li>
        <li>Daisy UI</li>
        <li>React JS</li>
        <li>Next JS</li>
        <li>Rechart</li>
    </ul>

<h2>Here is Listed key features of this app</h2>

<ul>
    <li>You can visit your friends profile</li>
    <li>You can interact your friends by Call, Message and Video Call </li>
    <li>You can visit timeline of interactions</li>
    <li>Also used pie chart to show which interaction is widely used by you </li>
</ul>
